# DN-JavaFSE-HandsOn

Digital Nurture 5.0 - Java Full Stack Engineer (React)

This repository contains week-wise hands-on exercises and assignments completed as part of the Cognizant Digital Nurture 5.0 Program.

## Structure

- Week-1 : Design Patterns, DSA, PL/SQL, TDD
- Week-2 : Spring Core, Maven, JPA, Hibernate
- Week-3 : Spring REST, Spring Boot
- Week-4 : SonarQube, Code Quality, Microservices
- Week-5 : React
- Week-6 : Advanced React
- Week-7 : Final Assignments
